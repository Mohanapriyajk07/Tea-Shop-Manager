# Tea-Shop-Manager
☕ Tea Shop Manager

Tea Shop Manager is a full-stack web application designed to help small tea shop owners manage their daily business operations digitally. Many small shops still track expenses and profits manually in notebooks, which can lead to errors or lost records. This app provides a simple and user-friendly system to register a tea shop, log daily stock investments such as milk, tea powder, snacks, and other items, and track financial performance through an interactive dashboard. The application focuses on simplicity so that even non-technical users can easily monitor revenue, profit, and loss trends in one place.

🚀 Features

Shop Registration
Register your tea shop using a mobile number and a 4-digit PIN.
Secure Login
Login using mobile number and PIN authentication.
Stock Investment Tracking

Record daily or monthly spending on:

Milk
Tea powder
Snacks
Sugar
Other supplies
Financial Dashboard
Visual charts showing:
Revenue trends
Profit and loss
Expense tracking
Simple User Interface
Designed to be easy for tea shop owners with no technical background.
Multi-User Access
Shop owners can share login credentials with trusted employees or family members to manage the shop together.

📊 How It Works

Register your tea shop using a mobile number and PIN.
Log in to the dashboard.
Record daily stock purchases and expenses.
Enter sales data.
View charts showing profit, loss, and revenue trends.

🎯 Project Goal

The goal of this project is to provide a simple digital management tool for small tea shops, helping owners track expenses, understand their business performance, and move away from manual bookkeeping.
